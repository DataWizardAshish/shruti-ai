# SHRUTI — Mission
**From Questions to Wisdom**

## Purpose
Make India's epic literature (Ramayana, Mahabharata) accessible through reflective, narrative-driven quizzes. Not trivia. Not speed tests. Wisdom through story.

## Core Belief
The epics aren't history — they're living guidance. Every question is a doorway into a story. Every answer revealed is a lesson earned.

## Target User
Curious Indians (and diaspora) who know the names but want to go deeper. No Sanskrit required. No prior knowledge gatekeeping. Start where you are.

## Product Principles
1. **Calm over competitive** — no timers, no leaderboards, no streaks pressure
2. **Story-first** — questions follow the narrative arc, not random shuffling
3. **Quality over quantity** — LLM-generated + human-reviewed, engagement-scored content
4. **Local-first** — progress stored on device, no account required to start

## v1 Definition of Done
User installs app → sees today's insight question → picks a story phase → answers questions → sees immersive explanation → tracks their journey arc. No login. No payment. No friction.

## Long-term Vision
- Ramayana complete → Mahabharata → Upanishads
- Hindi + regional languages
- Community-contributed questions (with LLM grounding validation)
- Personalized learning paths based on engagement patterns
